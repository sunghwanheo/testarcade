# testarcade 게임 개발자 가이드

## 전달 파일 목록

받으신 파일 중 개발에 필요한 것들입니다:

```
.claude/
└── skills/
    └── add-lang/
        └── SKILL.md    ← /add-lang 슬래시 커맨드
개발자가이드.md          ← 이 문서
```

`.claude/` 폴더를 **본인 게임 프로젝트 루트**에 그대로 복사하세요.

---

## 한/영 다국어 추가 방법

새 게임에 다국어를 추가할 때는 **Claude Code의 `/add-lang` 커맨드**를 사용하세요.
수동 작업 없이 자동으로 처리됩니다.

### 사전 준비

1. [Claude Code 설치](https://claude.ai/download) (없는 경우)
2. 전달받은 `.claude/` 폴더를 프로젝트 루트에 위치시키기

### 실행 방법

1. Claude Code에서 게임 폴더를 열기
2. 채팅창에 입력:
   ```
   /add-lang
   ```
3. Claude가 자동으로 아래 작업을 수행합니다:
   - `lang.js` 생성
   - `index.html`에 스크립트 태그 추가
   - 한/영 텍스트 매핑(`LANG` 객체) 작성
   - 하드코딩된 한국어 문자열 교체

---

## 수동으로 구현할 경우 (참고)

`/add-lang`을 사용하지 않고 직접 구현하는 경우의 패턴입니다.

### 1. lang.js 복사
다른 게임 폴더의 `lang.js`를 그대로 복사하세요. 수정 불필요.

### 2. index.html에 스크립트 추가
`<head>` 안 또는 첫 번째 `<script>` 바로 앞:
```html
<script src="./lang.js"></script>
```

### 3. LANG 객체 + applyLang 구현
게임 스크립트 최상단에 추가:
```javascript
const LANG = {
  ko: {
    startBtn: '시작',
    score:    '점수',
    // ... 게임에 나오는 모든 한국어 텍스트
  },
  en: {
    startBtn: 'Start',
    score:    'Score',
    // ...
  }
};

function applyLang(l) {
  const t = LANG[l];
  document.getElementById('start-btn').textContent = t.startBtn;
  // ... 텍스트가 있는 모든 DOM 요소 업데이트
}
```

### 4. 동적 텍스트 교체
JS 코드에서 한국어 문자열을 직접 쓴 부분을 교체:
```javascript
// 변경 전
el.textContent = '게임 시작!';

// 변경 후
el.textContent = LANG[getLang()].startBtn;
```

## lang.js 제공 함수

| 함수 | 설명 |
|------|------|
| `getLang()` | 현재 언어 반환 (`'ko'` or `'en'`) |
| `toggleLang()` | 언어 전환 (버튼 클릭 시 자동 호출) |
| `speakLang(koAudio, enText)` | 한국어는 오디오 재생, 영어는 TTS |

> 언어 토글 버튼은 `lang.js`가 자동으로 화면 오른쪽 상단에 주입합니다. HTML에 별도 추가 불필요.

> 언어 선택은 `localStorage`에 저장되어 게임 간에 유지됩니다.
